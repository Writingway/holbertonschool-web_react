# Holberton School - TypeScript

## Description
Ce dépôt regroupe les projets réalisés autour de **React** dans le cadre de la formation Holberton School.

Les projets couvrent les bases de React, la gestion des composants, l’état de l’application, ainsi que les bonnes pratiques du développement Front-End moderne.

## Objectifs
- Comprendre l’architecture React
- Créer des composants réutilisables
- Gérer l’état et les props
- Construire des interfaces dynamiques et modernes

## Technologies
- React
- JavaScript (ES6+)
- HTML5
- CSS3
- Node.js
